# **App Name**: TableTop Touch

## Core Features:

- QR Code Scanning: Scan QR codes on tables using the device's camera to identify the table number.
- Menu Display: Display the restaurant's menu, fetched from Firestore, categorized and searchable.
- Order Customization: Allow users to add items to their cart and customize quantities and special instructions.
- Realtime Order Tracking: Update the timeline to show the current status of the order.
- Staff Dashboard: Display a dashboard for staff to view live orders, update order statuses, and manage menu items, updating via Firestore.
- Image Handling: Store the food images in Firebase Storage.
- AI Wait Time Estimator: Employ a generative AI tool to predict the readiness of orders based on the items ordered and staff inputs, factoring in Firestore updates. Use AI to respond to call waiter requests. 

## Style Guidelines:

- Primary color: Dark background (#0A0A0C) for a luxurious feel.
- Secondary color: Slightly lighter dark background (#141416) to provide layers of contrast and separation.
- Accent color: Vibrant orange (#FF8C32) for primary interactive elements like buttons and highlights.
- Secondary accent: Warm yellow (#F0A826) to give visual interest on text elements and to further call the eye.
- Success color: Green (#2ECC71) for order status updates.
- Body and headline font: 'Inter' sans-serif for a clean, modern look.
- Use simple, minimalist icons with rounded corners to match the overall aesthetic.
- Employ rounded corners (20-26px) and soft shadows to create a modern, approachable interface.
- Incorporate smooth transitions and subtle micro-interactions to enhance user engagement.
